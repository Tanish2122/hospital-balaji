# Balaji Hospital Jaipur - Next.js Website

This is the standalone Next.js project for Balaji Hospital Jaipur, populated with the official content and structure.

## Installation

```bash
npm install
```

## Development

Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Production Build

Build production:

```bash
npm run build
```

Then to start the production server:

```bash
npm start
```
